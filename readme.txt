1.Install python 3.5, keras and Matlab
2.Download the all the files from  https://github.com/lungproject/Pancreas, and the trained model (Pancreas.hdf5) was in the folder named model.
3.Run Generatepathch.m with Matlab to generate the. npy file as the input of the model.
4.Run test_allpatient.py with python to generate the prediction results of all the input slices.
5.Run Obtainresults.m with Matlab to obtain the predicted DLS of each patient.
